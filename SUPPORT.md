# サポート方針

## 対象

正式リリース前は、最新のGitHub Releaseと `main` ブランチを対象にします。

## 問い合わせ先

- 不具合報告: GitHub Issues
- 機能要望: GitHub Issues
- 質問や使い方相談: GitHub Discussions
- 脆弱性報告: `SECURITY.md`

## 投稿しないでください

- 実データを含むSQLiteファイル。
- タスク名、サブタスク名、メモ本文、通知本文。
- 秘密情報、APIキー、トークン。
- 個人情報を含むスクリーンショット。
- OSユーザー名や端末固有パスが分かるログ。

## 既知制限

- MVPでは自動更新はありません。
- 署名なしRelease artifactでは、OSの警告が表示される場合があります。
- Linux配布、ストア配布、クラウド同期はMVP対象外です。
