# [UI] UIをDB接続へ切り替える

## 目的

現在のモック表示を、SQLite Repository経由の実データ表示へ切り替える。

## 対応内容

- タスクリストをDB由来にする。
- サブタスクリストをDB由来にする。
- 週カレンダーをDBクエリ由来にする。
- タイマー開始/停止ボタンをUse Caseへ接続する。
- エラー表示と空状態を実装する。

## 完了条件

- アプリ再起動後もタスクとサブタスクが残る。
- タイマー状態が再起動後に復元される。
- 週カレンダーがDBの予定を表示する。
- UI上でメモ本文をHTMLとして描画しない。

## 危険ケース

- 大量データを一括読み込みして週カレンダーが固まる。
- Tauri command失敗時にUIが無反応になる。
- 表示用ステータスとDBステータスがずれる。
