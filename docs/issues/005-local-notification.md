# [Notification] ローカル通知を実装する

## 目的

開始予定日・期限に対するローカル通知を、Windows/macOSで扱えるようにする。

## 対応内容

- 通知表示モード設定をDBへ保存する。
- `title_only` と `generic` の通知本文生成を実装する。
- OS通知アダプターを実装する。
- 通知登録成功/失敗状態を保存する。
- 通知登録失敗時の再試行導線を作る。

## 完了条件

- 通知表示モードを設定できる。
- `generic` ではタスク/サブタスクタイトルを通知に含めない。
- OS通知登録失敗がUIで分かる。
- macOS/Windowsの手動確認手順がある。

## 危険ケース

- `generic` 設定でもタイトルがOS通知に渡る。
- 通知登録失敗を握りつぶす。
- OSごとの通知権限差分を考慮しない。
