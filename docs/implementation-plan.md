# 実装計画

## Phase 0: 設計ベースライン

成果物:

- MVP仕様。
- アーキテクチャ資料。
- ドメインモデル。
- データベーススキーマ案。
- セキュリティ設計。
- レビューチェックリスト。
- GitHubテンプレートとリポジトリ設定。

完了条件:

- 設計レビューが完了している。
- `docs/mvp-spec.md` の未決事項が解決済み、または明示的に延期されている。

## Phase 1: プロジェクトスキャフォールド

成果物:

- Tauriプロジェクト構成。
- React + TypeScriptフロントエンド。
- Rust command境界。
- SQLiteマイグレーション実行器。
- 依存関係レビュー。

完了条件:

- macOS/Windowsの開発環境でアプリが起動する。
- 設定上、アプリ実行時の外部通信が無効化されている。
- 空データベースを初期化できる。
- 未決事項がプロダクト判断として反映されている。

現在のスキャフォールド状態:

- Tauri、Vite、React、TypeScriptのファイルは配置済み。
- Rust command境界はbootstrapコマンドのみ。
- SQLite初期マイグレーションは配置済み。
- 実行時HTTP、自動更新、分析、リモートアセット機能は設定していない。
