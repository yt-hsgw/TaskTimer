# MVP仕様

## 目的

Windows/macOSで動作するオフライン対応デスクトップタスクタイマーを作る。タスク計画、サブタスク計画、単一アクティブタイマー、ローカル通知、週カレンダーをMVP範囲とする。

## ユーザー

主なユーザー:

- 自分の業務タスクを整理し、実作業時間をローカルで記録したい個人の実務者。

MVPで扱わないこと:

- チーム共同編集。
- クラウド同期。
- モバイルアプリ。
- 外部カレンダー同期。
- Webログイン。
- 分析送信。
- アプリ実行時のネットワーク通信。

## 機能要件

### タスク

- ユーザーはタスクを作成、編集、完了、アーカイブ、削除できる。
- タスクはタイトル、状態、開始予定日、期限、メモ、並び順、作成日時、更新日時を持つ。
- タスクタイトルは必須。
- タスク期限日は開始予定日より前にできない。
- 完了したタスクは完了日時を記録する。
- 未完了サブタスクがある親タスクも、明示確認後であれば完了できる。
- タスク削除時は、タスク、子サブタスク、タイマー履歴、通知ルールを同一トランザクションでソフト削除する。

### サブタスク

- ユーザーはタスク配下のサブタスクを作成、編集、完了、アーカイブ、削除できる。
- サブタスクはタイトル、状態、開始予定日、期限、メモ、並び順、作成日時、更新日時を持つ。
- サブタスクタイトルは必須。
- サブタスク期限日は開始予定日より前にできない。
- サブタスクは親タスクとは独立したタイマー履歴を持てる。
- サブタスク削除時は、サブタスク、タイマー履歴、通知ルールを同一トランザクションでソフト削除する。

### タイマー

- ユーザーは1つのタスクまたは1つのサブタスクに対してタイマーを開始できる。
- アプリ全体で同時に開始できるタイマーは1件だけ。
- ユーザーはアクティブタイマーを停止できる。
- タイマー履歴は保持する。
- タイマー開始中にアプリを閉じた場合、再起動時は保存済みの `started_at` から経過時間を算出する。
- 完了済みまたはアーカイブ済みの対象はタイマー開始不可。

### 通知

- ユーザーはタスクまたはサブタスクの開始予定日・期限に対してローカル通知を設定できる。
- 通知設定はローカルに保存する。
- デフォルトの通知表示モードは対象タイトルのみを表示する。
- ユーザーは設定画面で通知表示モードを変更できる。
- 通知表示モードは `title_only` と `generic` をサポートする。
- OS通知登録はDBコミット後に実行する。
- OS通知登録に失敗した場合、アプリは再試行可能な登録状態を記録する。
- MVPでは、アプリ起動中または再読み込み時に期限到来済みの通知ルールをOS通知として送信する。
- 将来時刻にOSへ完全スケジュール登録する機能はMVP後の改善候補とする。

### 週カレンダー

- MVPのカレンダー画面は週表示とする。
- 週表示には、開始予定日と期限に基づいてタスクとサブタスクを表示する。
- 週表示には、タイマー実行中の対象を表示する。
- 週表示は前週・翌週へ移動できる。

## 非機能要件

- アプリ実行時の外部通信は禁止。
- Windows/macOSで動作する。
- データはローカルSQLiteに保存する。
- インターネット接続なしで利用できる。
- ユーザー内容を端末外へ送信しない。
- 5,000件のタスクと20,000件のサブタスクでもUI操作が固まらないことを目指す。

## 受け入れ条件

- ユーザーがタスクとサブタスクを作成できる。
- タスクとサブタスクの開始予定日・期限を設定できる。
- タスクタイマーを開始でき、停止するまで別タイマーを開始できない。
- サブタスクタイマーを開始でき、停止するまで別タイマーを開始できない。
- タイマー開始中にアプリを閉じて再起動しても、正しい経過時間を表示できる。
- 週カレンダーでタスクとサブタスクを確認できる。
- 未完了サブタスクがある親タスクは、確認後にだけ完了できる。
- タスク削除時に関連タイマー履歴がソフト削除される。
- 通知表示モードを設定画面で選べる。
- アプリ実行時にネットワーク通信しない。

## 決定事項

- 未完了サブタスクがある親タスクの完了時は確認メッセージを表示する。ユーザーが確認した場合、親タスクを完了できる。
- タスク削除時は、タスク、サブタスク、タイマー履歴、通知ルールをソフト削除する。履歴があることを理由に削除をブロックしない。開始中タイマーもソフト削除し、通常のアクティブタイマー検索から除外する。
- 通知表示は `title_only` をデフォルトとする。設定で `generic` に切り替えられる。
