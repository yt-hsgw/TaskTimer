# TaskTimer v0.1.0 Release Notes

この文書はGitHub Release `app-v0.1.0` の公開内容と一致させるためのRelease notesです。

## 公開状態

v0.1.0はWindows先行の通常Releaseとして公開済みです。

- GitHub Release: https://github.com/yt-hsgw/TaskTimer/releases/tag/app-v0.1.0
- 公開日: 2026-07-10
- 対象commit: `478f8d5932b6c85266cc0850912ac4561c6eb8c9`
- Windows実機確認: 2026-07-11に完了報告済み
- Release前静的監査: `npm run audit:runtime-privacy` 成功
- Windows runner検証: NSISサイレントインストールとサイレントアンインストール成功

macOS版はv0.1.0では提供しません。Apple Developer ID署名とApple公証を準備したReleaseで提供します。

## ダウンロード

GitHub Releasesから利用OSに合うartifactをダウンロードしてください。

- Windows: `TaskTimer_0.1.0_windows_x64-setup.exe`
- SHA-256: `ab1165b1b83541c5b2c909984a93b1129df9fca0817a40f2af11ae32f251f5cf`
- macOS Apple Silicon: v0.1.0では提供しません。
- macOS Intel: v0.1.0では提供しません。

Linux版、ストア配布、自動更新はv0.1.0の対象外です。

## 主な機能

- タスクとサブタスクの作成、完了、削除。
- タスクとサブタスクそれぞれの開始日、終了日、メモ。
- アプリ全体で同時に1件だけ開始できるタイマー。
- タイマーの開始、一時停止、再開、終了。
- サブタスク進捗、お気に入り、期限有無、完了セクションを持つタスク一覧。
- 選択したタスクを編集できる右詳細ペイン。
- 週表示カレンダー。
- 通知表示タイプの切り替え。
- オフライン前提のローカルSQLite保存。

## プライバシーと通信

- アプリ実行時の外部通信はありません。
- 自動更新はありません。新しいバージョンはGitHub Releasesで確認してください。
- タスク名、サブタスク名、メモ本文、通知本文はローカル端末内に保存されます。
- GitHub Issue、Discussions、PR、Releaseコメントへ、実データを含むSQLiteファイル、タスク名、メモ本文、通知本文、個人的なスクリーンショット、秘密情報を投稿しないでください。

## 既知制限

- Windows版はコード署名未設定です。Windows SmartScreenまたは組織ポリシーにより警告やブロックが表示される場合があります。GitHub Releasesの公式URL、Release notes、SHA-256を確認してから利用判断してください。
- macOS版はv0.1.0の配布対象外です。
- 自動更新はありません。
- Linux版は配布しません。
- `glib` advisoryはLinux限定依存であり、Windows配布物では未使用としてリスク受容しています。Linux artifactは配布せず、上流依存は週次監視します。判断はADR 0007に記録しています。
- OS通知の権限表示や表示名は、Windowsのバージョンやインストール状態により差が出る場合があります。

## 確認結果

- Windows実機:
  - インストール、起動、基本操作、アンインストールを確認済み。
  - 追加の公開ブロッカー報告なし。
- GitHub-hosted Windows runner:
  - NSISサイレントインストール成功。
  - NSISサイレントアンインストール成功。
- 静的監査:
  - `npm run audit:runtime-privacy` 成功。
  - 実行時外部通信、自動更新、リモートアセット、実行時ログ出力の入口がないことを確認。

## ロールバック判断

公開後に次のいずれかが見つかった場合は、Releaseを非公開化または削除し、Issueへ影響範囲と回避策を記録します。

- Windows版のインストール、起動、アンインストールができない。
- オフライン起動できない。
- タスク、サブタスク、タイマーの主要操作でデータ破損または復旧不能な失敗が起きる。
- 通知権限拒否時にアプリがクラッシュする。
- Release artifactに秘密情報、ローカルDB、ログ、個人データが含まれる。
- 既知制限として説明していない外部通信または自動更新が見つかる。
