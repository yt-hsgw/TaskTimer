## ダウンロード

GitHub Releasesから利用OSに合うartifactをダウンロードしてください。

- macOS Apple Silicon: `TaskTimer_0.1.0_darwin_aarch64.dmg`
- macOS Intel: `TaskTimer_0.1.0_darwin_x64.dmg`

Linux版、ストア配布、自動更新はv0.1.0の対象外です。

## 主な機能

- タスクとサブタスクの作成、完了、削除。
- タスクとサブタスクそれぞれの開始日、終了日、メモ。
- アプリ全体で同時に1件だけ開始できるタイマー。
- タイマーの開始、一時停止、再開、終了。
- サブタスク進捗、お気に入り、期限有無、完了セクションを持つタスク一覧。
- 選択したタスクを編集できる右詳細ペイン。
- 週表示カレンダー。
- 通知表示タイプの切り替え。
- オフライン前提のローカルSQLite保存。

## プライバシーと通信

- アプリ実行時の外部通信はありません。
- 自動更新はありません。新しいバージョンはGitHub Releasesで確認してください。
- タスク名、サブタスク名、メモ本文、通知本文はローカル端末内に保存されます。
- GitHub Issue、Discussions、PR、Releaseコメントへ、実データを含むSQLiteファイル、タスク名、メモ本文、通知本文、個人的なスクリーンショット、秘密情報を投稿しないでください。

## 既知制限

- Windows版はコード署名未設定です。Windows SmartScreenなどのOS警告が表示される場合があります。
- macOS版はDeveloper ID署名とApple公証済みartifactだけを公開対象にします。署名・公証が確認できないDraft Releaseは公開しません。
- 自動更新はありません。
- Linux版は配布しません。
- Dependabot alert #1はIssue #22で追跡しています。v0.1.0はLinux artifactを配布対象に含めません。
- OS通知の権限表示や表示名は、macOS/Windowsのバージョンやインストール状態により差が出る場合があります。

## ロールバック判断

次のいずれかが見つかった場合はDraft Releaseを公開しません。公開後に見つかった場合は、Releaseを非公開化または削除し、Issueへ影響範囲と回避策を記録します。

- macOSまたはWindowsでインストールできない。
- オフライン起動できない。
- タスク、サブタスク、タイマーの主要操作でデータ破損または復旧不能な失敗が起きる。

- Release artifactに秘密情報、ローカルDB、ログ、個人データが含まれる。
